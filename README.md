# jiaoben
===========

# Install

CentOS:

	yum -y install git
	git clone https://github.com/wangbingyan90/jiaoben.git
	./ssr.sh
	
Debian / Ubuntu:

	apt-get -y update && apt-get  -y install git
    git clone https://github.com/wangbingyan90/jiaoben.git && cd jiaoben && chmod +x ssr_ubuntu.sh && ./ssr_ubuntu.sh
